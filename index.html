Проекты

Примеры работ, сделанных в процессе изучения фронтенд-разработки.

Страница Креативного агентства

Использованные технологии: Flexbox для гибкого расположения элементов, Float для обтекания текста вокруг изображений и создания колонок, медиа-запросы для адаптивного дизайна.

Картина художника-абстракциониста

Использованные технологии: CSS Grid для создания сложной сетки, обеспечивающей точное управление размерами и расположением элементов.

Сайт с использованием Bootstrap

Использованные технологии: Bootstrap5, включая адаптивные колонки, фрагменты кода: heroes, features, jumbotron, pricing, footer; анимация градиента.

Игра на барабане

Использованные технологии: DOM, обработчики событий. Игра позволяет нажимать на изображения барабанных элементов или соответствующие клавиши для воспроизведения звуков и визуальных эффектов.

Simon Game

Использованные технологии: jQuery; обработчики событий. Игра включает в себя генерацию случайной последовательности сигналов, обработку ввода пользователя и сравнение его действий с ожидаемыми.

BookList

Использованные технологии: DOM, ООП, обработчики событий. Пользователь может добавлять книги в список и удалять книги из списка. Если какое-то поле не заполнено, книга не будет добавлена.
